# Landing Page Aniversário — One Piece: Elbaf Arc

Landing page temática sobre o arco de Elbaf de One Piece, com countdown, seções de conteúdo animadas e design responsivo.

## Tecnologias

- **HTML5**
- **SCSS** estilização modular com variáveis e componentes
- **JavaScript** countdown timer e animações
- **Parcel 2** bundler com compilação de SCSS
- **AOS** animações ao scroll

## Como utilizar

### Pré-requisitos

- [Node.js](https://nodejs.org/) instalado

### Instalação

```bash
npm install
```

### Desenvolvimento

```bash
npm run dev
```

Acesse `http://localhost:1234` no navegador.

### Build para produção

```bash
npm run build
```
